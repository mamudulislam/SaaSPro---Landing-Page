Bento Grid Customization

The features section uses a grid divided into 12 columns. You can change item spans using Bootstrap classes inside `style.css` custom classes:

<div class="bento-item large"> <!-- Spans 8 cols -->
<div class="bento-item small"> <!-- Spans 4 cols -->
<div class="bento-item full">  <!-- Spans 12 cols -->

3D Hero Mockup

The hero image automatically follows the mouse cursor with a 3D tilt effect. To adjust the sensitivity, open `js/main.js` and edit the divisor:

const xAxis = (window.innerWidth / 2 - e.pageX) / 50; // Change 50 to higher for LESS movement

Elite Theming System

All colors are controlled via CSS Variables. To change the brand identity, modify these core tokens in `style.css`:

:root {
    --p-indigo: #4F46E5; /* Main Brand */
    --p-violet: #7C3AED; /* Secondary */
    --p-teal:   #06B6D4; /* Accent */
}
